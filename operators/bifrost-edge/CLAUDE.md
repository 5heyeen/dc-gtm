# Bifrost Edge — Meeting Prep

This folder contains research and meeting preparation for Bifrost Edge.

## Setup

1. Read `operator.md` for known context.
2. Read `../../CLAUDE.md` for the full workflow.
3. Reference files at `../../metier/` and prompt templates at `../../templates/prompts/`.

## Commands

- **"Run all phases"** — Execute the full 6-phase research pipeline.
- **"Run phase N"** — Execute a specific phase.
- **"Update brief"** — Re-run phases 5-6 from existing research.

## Research Outputs

All outputs are saved to the `research/` subfolder.
